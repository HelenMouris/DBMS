# DBMS
A java implementation of a database management system with support for Octrees Indices
  
The engine supports:  
- Creating tables  
- Inserting tuples  
- Updating tuples  
- Deleting tuples  
- Searching tables  
- Creating an Octree index  
- Using the created octree indices where appropriate.
